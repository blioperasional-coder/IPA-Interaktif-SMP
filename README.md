# IPA Interaktif SMP

Media pembelajaran IPA SMP interaktif — **Biologi**, **Kimia**, dan **Tata Surya** —
dalam satu halaman web. Berisi materi ringkas plus simulasi interaktif (uji makanan,
siklus menstruasi, peredaran darah, lab asam-basa, orbit planet, fase bulan, gerhana, dll).

▶️ **Buka:** https://blioperasional-coder.github.io/IPA-Interaktif-SMP/

Dibuat sebagai bahan belajar yang menyenangkan & mudah dipahami. Satu file HTML,
berjalan offline, tanpa library eksternal.
